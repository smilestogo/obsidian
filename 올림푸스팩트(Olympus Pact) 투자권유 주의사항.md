# 올림푸스팩트(Olympus Pact) 투자권유 주의사항

## 📋 기본 정보

### 올림푸스팩트란?
- **프로젝트명**: Olympus Pact
- **자칭**: "차세대 superchain 금융 시스템" (OlympusDAO 핵심 개발자들이 만든다고 주장)
- **출현 시기**: 2025년 5월경부터 Medium 등에 마케팅 글 게시
- **특징**: 
  - OlympusDAO(2021년 DeFi 프로젝트)의 후속 프로젝트라고 주장
  - OlympusDAO는 한때 40억 달러 규모였으나 고인플레이션 구조로 인해 문제 발생
  - "3년간의 반성과 재구축"을 강조하며 등장


자 이걸 테스트해보자고. 


### ⚠️ 중요 경고사항
1. **일본 의료기기 회사 "올림푸스(Olympus Corporation)"와는 전혀 무관**
2. 공식 웹사이트 접속 불가 (조사 시점 기준)
3. 한국 내 직접적인 사기 피해 사례는 아직 보도되지 않았으나, **암호화폐/블록체인 투자권유는 사기 가능성 높음**

## 🚨 투자권유 상담 시 필수 체크리스트

### 1. 사업자 정보 확인
- [ ] **금융투자업 등록 여부** 확인
  - 금융감독원 금융소비자정보 포털 '파인(Fine)' 검색
  - URL: https://fine.fss.or.kr
  - 미등록 업체일 경우 즉시 투자 중단

- [ ] **회사 법인등록증** 요구
  - 실제 법인이 존재하는지 확인
  - 사업자등록번호로 국세청 홈택스에서 조회

- [ ] **대표자 신원** 확인
  - 실명과 신분증 확인
  - OlympusDAO 핵심 개발자 출신이라는 주장의 증거 요구

### 2. 투자 구조 검증

#### 🔴 즉시 사기로 의심해야 할 표현들
- "원금 보장" / "손실 제로"
- "월 수익 10% 이상 보장"
- "100% 안전한 투자"
- "지금 아니면 기회 없음" (긴급성 강조)
- "VIP 회원만 가능" (희소성 강조)
- "OlympusDAO 초기 투자자들이 350배 수익" (과거 성공 사례 강조)

#### 📌 반드시 질문해야 할 사항
1. **수익 구조가 명확한가?**
   - "어떻게 수익이 발생하는가?"
   - "구체적인 투자처는 어디인가?"
   - 모호한 답변 시 → 사기 가능성 높음

2. **출금 조건은?**
   - "언제든지 출금 가능한가?"
   - "출금 수수료나 제한사항은?"
   - "락업 기간이 있는가?"
   - 출금이 어렵거나 조건이 복잡 → 전형적인 폰지사기 패턴

3. **백서(White Paper) 제공 여부**
   - 기술 문서가 있는가?
   - 내용이 구체적이고 검증 가능한가?
   - 전문 용어만 나열하고 실체 없음 → 사기 가능성

### 3. 권유자 정보 파악

- [ ] **권유자와의 관계**
  - 지인인 경우: 지인도 피해자일 가능성 (다단계 구조)
  - SNS/온라인에서 접근: 조직적 사기 가능성 높음

- [ ] **수익 인증 요구**
  - 실제 출금 가능한 수익인지 확인
  - 화면 캡처만으로는 조작 가능 → 실제 출금 확인 필수

- [ ] **다른 투자자 정보**
  - 실제 투자자가 몇 명인가?
  - 투자자들끼리 직접 연락 가능한가?
  - 바람잡이(사기조직 멤버)일 가능성 염두

### 4. 암호화폐 특성 관련 질문

#### 반드시 확인할 기술적 사항
1. **토큰 정보**
   - [ ] 토큰 이름과 심볼
   - [ ] 어느 블록체인에서 발행되는가? (이더리움, 솔라나 등)
   - [ ] 토큰 컨트랙트 주소 제공 가능한가?
   - [ ] Etherscan/CoinGecko 등에서 검증 가능한가?

2. **거래소 상장 여부**
   - [ ] 어느 거래소에 상장되어 있는가?
   - [ ] 국내 원화 거래소(업비트, 빗썸 등)에 상장되어 있는가?
   - [ ] 상장 예정이라는 주장의 증거는?

3. **지갑 관리**
   - [ ] 자체 지갑 설치를 요구하는가? (피싱 위험)
   - [ ] 개인키나 시드 문구를 요구하는가? (절대 제공 금지)

### 5. 법적 리스크 확인

- [ ] **계약서 제공 여부**
  - 서면 계약서가 있는가?
  - 계약서 내용에 책임 회피 조항은 없는가?

- [ ] **특금법 준수 여부** (한국 특정금융정보법)
  - 가상자산사업자 신고 완료 여부
  - 실명 계좌 사용 여부
  - ISMS 인증 여부

## 🚩 전형적인 암호화폐 사기 패턴

### 1. 폰지 사기 (Ponzi Scheme)
- 신규 투자자 돈으로 기존 투자자에게 배당
- 초기에는 수익이 나는 것처럼 보임
- 신규 투자자 모집이 멈추면 붕괴

### 2. 출구 사기 (Exit Scam)
- 일정 금액 모금 후 잠적
- 웹사이트 폐쇄, 연락 두절

### 3. 피그 부처링 (Pig Butchering)
- 로맨스 스캠과 결합
- 장기간 신뢰 구축 후 투자 유도
- "프라이빗 코인 세일" 등의 명목

### 4. 가짜 거래소/지갑
- 자체 제작한 거래 플랫폼 설치 유도
- 실제로는 거래 없이 숫자만 조작
- 출금 시 "수수료", "세금", "인증비" 등 추가 요구

### 투자 금지 신호 (하나라도 해당되면 즉시 중단)
1. ⛔ 금융감독원 등록되지 않은 업체
2. ⛔ 원금 보장이나 고수익 보장 약속
3. ⛔ 출금이 자유롭지 않거나 조건부
4. ⛔ 투자 구조 설명이 모호하거나 이해 불가
5. ⛔ 긴급성을 강조 ("지금 아니면 안 됨")
6. ⛔ 다단계 구조 (지인 추천 시 보상)
7. ⛔ 검증되지 않은 토큰이나 자체 플랫폼
8. ⛔ 해외(특히 동남아) 운영 의심
9. ⛔ SNS/텔레그램 등 비공식 채널 주요 소통 수단
10. ⛔ 유명인 사칭 또는 AI 딥페이크 사용
